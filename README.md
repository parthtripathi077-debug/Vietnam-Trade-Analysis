# Vietnam Trade Analysis

## Overview

This project examines Vietnam’s trade dynamics by combining labour market data (ILOSTAT) with macroeconomic indicators (World Bank). The aim is to understand how structural changes in the economy have influenced Vietnam’s export-led growth.

## Approach

The analysis uses Python (Pandas, NumPy, Matplotlib) to study trends in GDP, trade openness, and sectoral employment over time. The results are interpreted using economic theories such as comparative advantage and structural transformation.

## Key Insights

Vietnam has experienced a clear shift from agriculture to industry and services, indicating structural transformation towards higher productivity sectors. At the same time, exports have grown significantly as a share of GDP, showing strong integration into global trade. Rising imports alongside exports suggest dependence on intermediate goods, highlighting Vietnam’s role in global value chains. Overall, the data reflects sustained economic growth supported by trade expansion.
